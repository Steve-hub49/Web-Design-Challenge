Summary: Latitude vs. X


This project analyzed how weather changes as one gets closer to the equator. To accomplish this, data was pulled from the OpenWeatherMap API to assemble a dataset on over 500 cities.
After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs. latitude. Several factors were considered including temperature, cloudiness, wind speed, and humidity.
This site provides the source data and visualizations created as part of the analysis.
